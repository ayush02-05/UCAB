Design a modern and responsive UI for a Cab Booking Web Application called "UCab", similar to Uber or Ola Cabs.

The design should be clean, minimal, and mobile responsive with card-based layouts and modern UI components.

The application has two main roles: User and Driver.

Use a dashboard style interface, modern color palette, and clear booking workflow.
